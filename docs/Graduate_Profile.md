# Graduate Profile

A graduate of TalentOS should be capable of independently:

- Understanding business problems
- Asking clarification questions
- Writing a PRD
- Defining user stories
- Defining acceptance criteria
- Designing solutions
- Using AI effectively
- Building software
- Testing software
- Deploying applications
- Creating documentation
- Presenting solutions
- Assessing production readiness

Graduates are not expected to review other engineers' code.

The objective is to create independent engineers, not senior engineers.
