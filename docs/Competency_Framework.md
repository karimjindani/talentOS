# Competency Framework

## Competencies

### 1. Problem Discovery

Understand business objectives.

### 2. Requirements Engineering

Convert ideas into requirements.

### 3. Solution Design

Design maintainable systems.

### 4. AI Collaboration

Work effectively with AI.

### 5. Software Construction

Build production-grade software.

### 6. Quality Engineering

Validate and test solutions.

### 7. Deployment & Operations

Deploy and troubleshoot software.

### 8. Documentation

Create maintainable documentation.

### 9. Communication

Present and defend solutions.

### 10. Professionalism

Work independently.

### 11. Engineering Leadership

Plan and organize work.

### 12. Production Readiness

Evaluate deployment readiness.
