## Overview

Regression tests ensure that the AI Mentor feature does not break any existing functionality. These follow the project's SSDLC principle: *"Every iteration of development must ensure that previously committed and tested work remains functional"* (`docs/sdlc.md`).

Two layers:
1. **Unit regression** — fast Vitest coverage (`npm.cmd run regression:unit`)
2. **Scenario regression** — end-to-end journeys (`npm.cmd run regression:<area>`)
