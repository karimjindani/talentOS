## Reference Endpoints (admin app)

The admin app exposes `POST /api/files/presign-upload`, `POST /api/files/[id]/confirm`, and
`GET /api/files/[id]/download`. All require an authenticated admin session (enforced by middleware) and
resolve the tenant from the host; the download route is tenant-scoped (a file id from another tenant
returns 404). These exercise the foundation and are reusable by the applicant app for CV-on-apply later.
