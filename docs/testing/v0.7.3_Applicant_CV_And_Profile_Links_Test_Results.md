## Notes

- The CV upload uses the **server-action proxy** model (apply action streams the file to MinIO via
  `putObject`) per `D-045`, not v0.7.0's browser-direct presigned upload.
- **Known limitation:** if the application insert fails after `putObject`, the uploaded object is
  orphaned in MinIO (no lifecycle cleanup yet) — consistent with v0.7.0 scope.
- The Playwright harness used for the interactive run lives outside the repo (scratch dir); no test
  tooling was added to the codebase.
