## Root cause evidence (from Keycloak logs)

```
ERROR [org.keycloak.services.error.KeycloakErrorHandler] Uncaught server error:
java.lang.ArithmeticException: / by zero
    at org.keycloak.models.utils.TimeBasedOTP$Clock.getCurrentInterval(TimeBasedOTP.java:132)
    at org.keycloak.models.utils.TimeBasedOTP.validateTOTP(TimeBasedOTP.java:82)
    at org.keycloak.authentication.requiredactions.UpdateTotp.processAction(UpdateTotp.java:94)
```

`getCurrentInterval` divides by `otpPolicyPeriod`; the realm omitted it, so Keycloak used `0`.
