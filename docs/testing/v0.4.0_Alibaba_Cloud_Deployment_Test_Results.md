## Notes

- This is a public-IP validation deployment, not a production-ready release.
- Production hardening will require HTTPS/domain routing, Keycloak production mode, managed database
  evaluation, backups, monitoring, and tighter administrative access controls.
- No ECS instance or public validation URL exists yet because Alibaba Cloud requires card/security
  verification before allowing instance creation.
