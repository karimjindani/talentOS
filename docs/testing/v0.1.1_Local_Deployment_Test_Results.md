## Notes

The Docker build initially failed because Next.js attempted to download SWC during image build. `v0.1.1` fixes this by pinning the Linux SWC package inside the Docker build and switching the web image to a Debian-based Node image.

The Docker build also warned that Prisma could not detect OpenSSL in the image. The build and runtime still completed successfully. This should be reviewed in a future hardening iteration.

`npm audit` continues to report 2 moderate dependency findings. `npm audit fix --force` was not applied because it can introduce breaking dependency changes.
