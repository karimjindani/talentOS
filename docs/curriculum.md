# TalentOS Curriculum

## Overview

TalentOS is a four-week AI-Native Software Engineering Apprenticeship Program.

The curriculum follows the Spiral Engineering Method (SEM).

Every week participants perform the complete engineering lifecycle:

1. Discover
2. Analyze
3. Specify
4. Design
5. Build
6. Test
7. Deploy
8. Present
9. Reflect
10. Production Readiness Review

The complexity of assignments increases each week.
