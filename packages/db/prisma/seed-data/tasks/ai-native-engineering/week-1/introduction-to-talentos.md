# Introduction to TalentOS

## What TalentOS Is

TalentOS is a four-week, mission-based internship platform. It combines guided learning tasks, practical missions, daily Engineering Journal reflection, evidence-based submission, and staff review.

## The Four-Week Journey

- Each week has learning and setup tasks tied to the program week.
- Each applicant receives a mission assignment for the week.
- The Engineering Journal records daily work, challenges, solutions, learning, AI usage, confidence, time, and evidence.
- A mission submission contains public repository, deployment, and Loom walkthrough links.
- Review feedback may accept the work, request revision on the same attempt, or create a fresh repeat attempt.

## Environment Setup Objective

- Install the tools required by the repository documentation.
- Clone TalentOS on a feature branch rather than working directly on main.
- Start the required local services.
- Open the Applicant Portal and confirm that login and dashboard navigation work.
- Record useful evidence and reflect on the setup process in the Engineering Journal.

Evidence and reflection matter because they show not only what was delivered, but also how the applicant reasoned, learned, and improved.
